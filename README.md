# Regula CocoaPods Specs

This repository contains CocoaPods podspecs for Regula iOS SDKs.

A podspec describes a specific version of a CocoaPods package, including where the SDK artifacts are downloaded from, which platforms are supported, and how the package should be integrated into an Xcode project.

This repository is intended to be used as a CocoaPods specs source. The SDK binaries and source files are not stored in this repository. Each podspec points CocoaPods to the corresponding SDK artifact.
